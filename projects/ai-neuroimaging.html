AI for Neuroimaging

We explore foundation‑model style pretraining for neuroimaging to improve downstream tasks such as risk prediction and lesion quantification.

Status: Active
Primary domain: Neuroimaging · Self‑supervised learning
Contact: chenm19@wfu.edu

Goals

  • Pretrain encoders with self‑supervised objectives on large unlabeled imaging datasets.
  • Evaluate transfer learning on Alzheimer’s‑related tasks.
  • Establish data quality checks and standardized preprocessing.

Getting Started

  1. Set up compute environment and storage with version control for large files (e.g., git‑LFS).
  2. Implement baseline SSL method and training loop.
  3. Define downstream tasks and metrics; prepare evaluation scripts.
