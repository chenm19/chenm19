Disease Prediction with Multi‑Modal Data

This student‑led project focuses on predicting Alzheimer’s disease onset and progression using multi‑modal data: neuroimaging (MRI/PET), omics (genomics, transcriptomics), and clinical measurements. We will evaluate classical ML and deep learning approaches with rigorous validation.

Status: Open for student development
Primary domain: Alzheimer’s disease · Predictive modeling
Contact: chenm19@wfu.edu

Goals

  • Assemble a clean training dataset with harmonized features and ethical approvals where required.
  • Benchmark baselines (logistic regression, random forest, XGBoost) vs. deep models (tabular transformers, multimodal fusion).
  • Calibrate and explain models (reliability curves, SHAP/attribution) for clinical interpretability.

Getting Started

  1. Survey candidate datasets (e.g., ADNI) and define inclusion criteria.
  2. Set up a reproducible pipeline (conda/poetry, data versioning, unit tests).
  3. Create evaluation protocol (train/val/test splits, external validation, metrics: AUROC, AUPRC, calibration error).
